# Fractal-Lab

An interactive web-based fractal visualisation project for exploring self-similar sets, affine maps, iterated function systems, complex dynamics, plane iterations, dendrites, fractal curves and 3D fractals.

This project is designed for teaching, experimentation and mathematical exploration. Each sectio nuses live canvas-based visualisations with adjustable controls, allowing users to zoom, pan, change parameters, estimate dimension and export generated images.

## Features
- Interactive fractal visuaslisations
- Mandelr=brot and Julia set rendering and parameter exploration
- Newton method basin visualisation
- Affine map and IFS construction tools
- Fractal and space-filling curve generators
- Plane iterations and attractor exploration
- DLA (diffusion-limited aggregation) dendrite simulation
- 3D fractal projection tools
- Dimension estimation in selected sections
- PNG exports, video recording, shared navigation and footer, canvas controls and light/dark theme support

## Lab Sections
### Home Page
The home page introduces the proejct and provides links to each lab section; acts as the main entry point for the visualiser.

Includes: "index.html"

### Complex Dynamics
These sections explore repeated calculations in the complex plane, including the Mandelbrot set and Julia set.

The Mandelbrot-Julia connection allows users to select a point in the Mandelbrot set and immediately seeing the corresponding Julia set.

The other section explores the Newton method basins of attraction.

Includes:
- "complex.html"
- "newtonMethod.html"

### Affine Maps and Iterated Function Systems
These labs explore fractals generated by repeated geometric transformations. Users cna experiment with scaling, rotation, translation, shearing and probability to build self-similar sets.

The affine section includes draggable geometric controls, while IFS supports deterministic and chaos-game constructions.

Includes:
- "affine.html"
- "ifs.html"

### Fractal and Space-Filling Curves
This lab explores curves generated by recursive rewriting systems and turtle graphics. Users can adjust iteration depth, drawing style, animation speed and custom rewriting rules.

Includes: "curves.html"

### Plane Iterations
This section visualises orbits of two-dimensional maps. Users enter function for 'x' and 'y', then observe how a grid of starting points evolve under repeated iteration.

The resulting images can reeeal attractors, spirals, orbit structures, and other dynamical patterns.

Includes: "planeIts.html"

### Stochastic Growth
This lab uses diffusion-limited aggregation to simulate branching fractal growth. Random walkers move through the plane and stick to an existing cluster, forming dendrite-like structures over time.

Users can control simulation resolution, walker behaviour, stick probability, seed type, colouring and animation.

Includes: "dendrites.html"

### 3D Fractals
This lab extends fractal exploration into three dimension; rendering projected views of 3D fractal points clouds and allowing users to rotate, pan, zoom, animate and export the result.

Includes: "3DFractals.html"

## Shared Tools
Each fractal lab uses shared interface tools:
- Wheel zoom ancd drag-to-pan
- Mathematical coordinate readouts
- Reset view controls

The reusable View2D system handles 2D canvas behaviour, while shared export utilities handle image and video downloads.
